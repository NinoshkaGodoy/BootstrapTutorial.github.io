Componentes de Bootstrap

Elementos reutilizables listos para usar en tus proyectos

¿Qué son los Componentes?

Los componentes son elementos de interfaz pre-construidos que puedes usar directamente en tus proyectos. Bootstrap incluye más de 20 componentes diseñados y probados para funcionar en cualquier dispositivo.

Tip: Todos los componentes son responsivos por defecto y se adaptan automáticamente a diferentes tamaños de pantalla.

Accordion (Acordeón)

Panel colapsable perfecto para FAQs y contenido organizado.

Bootstrap es un framework CSS de código abierto que facilita el desarrollo de interfaces web responsivas y modernas.

Sí, Bootstrap es muy intuitivo. Solo necesitas conocimientos básicos de HTML y CSS para comenzar a usarlo.

Completamente gratuito y de código abierto. Puedes usarlo en proyectos personales y comerciales sin restricciones.
<div class="accordion" id="accordionExample">
  <div class="accordion-item">
    <h2 class="accordion-header">
      <button class="accordion-button" type="button" data-bs-toggle="collapse" data-bs-target="#collapseOne">
        Título del Accordion
      </button>
    </h2>
    <div id="collapseOne" class="accordion-collapse collapse show">
      <div class="accordion-body">Contenido...</div>
    </div>
  </div>
</div>

Tooltips & Popovers

Información contextual que aparece al pasar el mouse o hacer clic.

Tooltips (al pasar el mouse)
Popovers (al hacer clic)
<!-- Tooltip -->
<button type="button" data-bs-toggle="tooltip" title="Texto del tooltip">
  Hover me
</button>

<!-- Popover -->
<button type="button" data-bs-toggle="popover" data-bs-title="Título" data-bs-content="Contenido aquí">
  Click me
</button>

<!-- JavaScript necesario -->
<script>
// Activar tooltips
var tooltips = document.querySelectorAll('[data-bs-toggle="tooltip"]');
tooltips.forEach(t => new bootstrap.Tooltip(t));

// Activar popovers
var popovers = document.querySelectorAll('[data-bs-toggle="popover"]');
popovers.forEach(p => new bootstrap.Popover(p));
</script>

<!-- JavaScript para mostrar -->
<script>

Tips para usar Componentes
  • Combina múltiples componentes para crear interfaces complejas
  • Usa las clases de utilidad para personalizar colores y espaciado
  • Los componentes son responsive por defecto
  • Algunos requieren JavaScript (modales, tooltips, etc.)
  • Consulta siempre la documentación oficial para opciones avanzadas
  • Puedes personalizar con CSS custom
