Utilidades de Bootstrap

Bootstrap proporciona una serie de utilidades rápidas basadas en clases que permiten aplicar estilos comunes sin necesidad de escribir CSS adicional. A continuación, un resumen en tabla y ejemplos prácticos abajo.

Utilidad Descripción Ejemplo
Spacing Márgenes y paddings con clases m-* y p-* (0–5). class="m-3" / class="px-4"
Display Controla la propiedad display con clases utilitarias (d-block, d-flex, etc.). class="d-flex"
Flexbox Utilidades para estructurar con flexbox: alineación y distribución. class="d-flex justify-content-between"
Colores Clases para texto y fondo como text-* y bg-*. class="text-white bg-success"
Bordes Clases para añadir/quitar bordes y bordes redondeados. class="border rounded"
Sombras Niveles de sombra con shadow, shadow-sm, shadow-lg. class="shadow-lg"
Tamaños Clases para anchuras/alturas relativas como w-25, w-50, h-100. class="w-50"
Tipografía Atajos para peso, estilo y transformación de texto (fw-bold, fst-italic, text-uppercase). class="fw-bold fst-italic"

Debajo tienes ejemplos interactivos que muestran estas utilidades aplicadas en tarjetas.
