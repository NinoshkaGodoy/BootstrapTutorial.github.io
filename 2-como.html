¿Cómo usar Bootstrap?

Guía paso a paso para empezar con Bootstrap

Métodos de Instalación

Existen varias formas de agregar Bootstrap a tu proyecto. Elige la que mejor se adapte a tus necesidades.

Método 1: CDN (Recomendado para principiantes)

La forma más rápida y sencilla. Solo agrega estos enlaces en tu archivo HTML:

Ventajas: No requiere descarga, siempre actualizado, carga rápida desde servidores globales
Paso 1: Crea tu archivo HTML básico
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Mi Proyecto Bootstrap</title>
</head>
<body>
  <h1>Hola Mundo</h1>
</body>
</html>
Paso 2: Agrega el CSS de Bootstrap en el <head>
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">
Paso 3: Agrega el JavaScript antes de cerrar </body>
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
Código completo:
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Mi Proyecto Bootstrap</title>
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
  <div class="container mt-5">
    <h1 class="text-primary">¡Hola Bootstrap!</h1>
    <button class="btn btn-success">Mi Botón</button>
  </div>
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>

Método 2: Descarga de Archivos

Descarga Bootstrap y guárdalo localmente en tu proyecto.

Ventajas: Funciona sin conexión a internet, control total sobre los archivos
Pasos:
  1. Descarga Bootstrap desde getbootstrap.com
  2. Extrae los archivos en una carpeta de tu proyecto (ej: assets/bootstrap/)
  3. Vincula los archivos en tu HTML:
    <link href="assets/bootstrap/css/bootstrap.min.css" rel="stylesheet">
    <script src="assets/bootstrap/js/bootstrap.bundle.min.js"></script>

Método 3: NPM (Para proyectos avanzados)

Si trabajas con Node.js y herramientas de build:

Nota: Requiere conocimientos de Node.js y npm
npm install bootstrap

Luego importa en tu JavaScript o Sass:

import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.min.js';

Primeros Pasos con Bootstrap

1. Estructura Básica

Siempre usa un contenedor para organizar tu contenido:

Código:
<div class="container">
  <h1>Mi Título</h1>
  <p>Mi contenido</p>
</div>
Resultado:
Mi Título

Mi contenido

2. Usa las Clases de Bootstrap

Bootstrap funciona mediante clases CSS. Ejemplos:


badge bg-info badge bg-warning
<button class="btn btn-primary">Mi Botón</button>
<span class="badge bg-info">Insignia</span>

3. Sistema de Grid

Crea layouts responsivos con el sistema de columnas:

col-md-4
col-md-4
col-md-4
<div class="row">
  <div class="col-md-4">Columna 1</div>
  <div class="col-md-4">Columna 2</div>
  <div class="col-md-4">Columna 3</div>
</div>

4. Utilidades de Espaciado

Controla márgenes y padding fácilmente:

p-3 = padding de 1rem
mt-4 = margin-top de 1.5rem
mx-auto = centrado horizontal
m-{tamaño} = margin
p-{tamaño} = padding
mt-3 = margin-top
mb-4 = margin-bottom
px-5 = padding horizontal (left + right)
py-3 = padding vertical (top + bottom)

Tips Importantes

Haz (Do's)
  • Usa la documentación oficial
  • Combina clases para personalizar
  • Usa el inspector del navegador
  • Prueba en diferentes dispositivos
  • Aprende el sistema de grid primero
No Hagas (Don'ts)
  • No modifiques los archivos de Bootstrap directamente
  • No uses IDs para estilos, usa clases
  • No olvides la etiqueta viewport
  • No mezcles versiones de Bootstrap
  • No uses inline styles si puedes evitarlo
