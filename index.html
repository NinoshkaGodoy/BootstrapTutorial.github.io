¿Qué es Bootstrap?

El framework CSS más popular del mundo

Definición

Bootstrap es un framework front-end de código abierto para diseñar sitios web y aplicaciones web responsivas y móviles.

Fue desarrollado originalmente por Twitter en 2011 y actualmente es mantenido por la comunidad de código abierto. Bootstrap proporciona un conjunto de herramientas CSS y JavaScript pre-construidas que facilitan la creación de interfaces de usuario consistentes y atractivas.

Características Principales

Diseño Responsivo

Sistema de grid flexible que se adapta automáticamente a diferentes tamaños de pantalla (móvil, tablet, desktop).

Componentes Pre-construidos

Incluye botones, formularios, navbars, cards, modales y muchos más componentes listos para usar.

Personalizable

Puedes personalizar colores, espaciados y estilos usando variables CSS o Sass.

Gran Comunidad

Amplia documentación, tutoriales y soporte de una comunidad global de desarrolladores.

Componentes de Bootstrap

Sistema Grid

Sistema de 12 columnas para layouts responsivos

Utilidades CSS

Clases para espaciado, colores, texto y más

JavaScript

Plugins para modales, tooltips, dropdowns

Historia Breve

  • 2011: Lanzado por Twitter como "Twitter Bootstrap"
  • 2012: Bootstrap 2 - Mayor expansión y adopción
  • 2013: Bootstrap 3 - Diseño "mobile-first"
  • 2018: Bootstrap 4 - Migración a Flexbox
  • 2021: Bootstrap 5 - Eliminación de jQuery, mejoras CSS
